## 程序说明
### 异环钓鱼自动化程序

**功能说明**
* 设置屏幕分辨率以适应不同设备
* 通过截图自动获取钓鱼进度条检测区域和模拟鼠标点击区域（可选临时保存或者永久保存）
* 设置模拟鼠标和键盘输入延迟，模拟真人输入
* 自动检测移动条和光标位置，模拟键盘输入，控制移动光标，使移动条和光标重合
* 模拟鼠标点击钓鱼按钮
* 自定义设置快捷键（F1~F12)
* 设置内存大小，避免过多占用系统资源
* 可保存切换配置文件


**使用方法**
1. 默认按F1截取鼠标点击区域获取坐标（本次有效，可自定义）
2. 默认按F2截取进度条区域获取坐标（本次有效，可自定义）
3. 默认按F3截取鼠标点击区域获取坐标（永久有效，可自定义）
4. 默认按F4截取进度条区域获取坐标（永久有效，可自定义）
5. 默认按F5开始/暂停程序（可自定义）
6. 默认按F6退出配置文件（可自定义）

* 默认按F7切换配置文件（可自定义）
* 默认按F8修改延迟，内存与颜色（可自定义）

**注意**
* 该程序需要管理员权限运行
* 默认配置1为2k屏幕，默认配置2为1080p屏幕
* 内存大小，输入延迟，移动条和光标不推荐修改
